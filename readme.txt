AI Wow Plugin utilizes OpenAI's API to generate key takeaways for your WordPress posts.

== Description ==

AI Wow Plugin helps WordPress site owners automatically generate key takeaways for their posts using OpenAI's powerful language models. By leveraging the AI, users can provide their readers with concise summaries, enhancing readability and engagement.

Features include:
- Generate key takeaways directly in the WordPress post editor.
- Customize where the key takeaways appear within your post content.
- Manage your OpenAI API key within the WordPress admin area.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-wow-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->AI Wow Plugin screen to configure the plugin.
4. Enter your OpenAI API key in the provided settings field.
5. Start generating key takeaways for your posts in the post editor.

== Frequently Asked Questions ==

= Do I need an OpenAI account to use this plugin? =

Yes, you need to sign up for an OpenAI account and obtain an API key to use this plugin.

= Is there a cost associated with using this plugin? =

While the AI Wow Plugin is free, using the OpenAI API may incur costs based on their pricing model. Please check OpenAI's pricing details for more information.

= Can I customize the style of the key takeaways? =

Yes, the key takeaways are wrapped in a distinct HTML element which can be styled with custom CSS.
